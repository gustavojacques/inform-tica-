# Aulas de Informática

## Curso: Gestão da Produção Industrial

### Objetivos
- Compreender os conceitos fundamentais da informática aplicados à gestão da produção.
- Desenvolver habilidades em ferramentas tecnológicas para otimização de processos.
- Analisar dados e tomar decisões informadas através da tecnologia.

### Conteúdo Coberto
1. Introdução à Informática
2. Sistemas de Informação
3. Softwares de Gestão
4. Análise de Dados
5. Tecnologias Emergentes na Indústria

### Ferramentas Utilizadas
- Microsoft Excel
- Google Sheets
- Softwares de Gestão Empresarial (ERP)
- Plataformas de Análise de Dados
